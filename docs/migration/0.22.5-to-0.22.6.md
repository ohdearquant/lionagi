# Migrating from 0.22.5 to 0.22.6

## Breaking changes

| Change | Impact | Error if you skip | ¬ß |
|--------|--------|-------------------|---|
| `branch.instruct()` removed | **High** | `AttributeError: 'Branch' object has no attribute 'instruct'` | ¬ß1 |
| `lionagi.operations.instruct` removed | **High** | `ModuleNotFoundError` | ¬ß2 |
| Persistence: `logs/agents/` ‚Üí `runs/{run_id}/` | **Medium** | Resume finds nothing | ¬ß3 |
| `last_branch.json` schema | **Low** | None (auto-normalized) | ¬ß4 |
| Flat artifact dumps removed | **Medium** | `FileNotFoundError` on old paths | ¬ß5 |
| DAG op name `"instruct"` ‚Üí `"operate"` | **Medium** | Orchestrator ignores op | ¬ß6 |
| `"codex/gpt-5.4"` fallback removed | **Low** | `ValueError: Provider must be provided` | ¬ß7 |
| `TEAM_WORKER_SYSTEM` composition | **Low** | Wrong prompt composition (silent) | ¬ß8 |

---

## ¬ß1 `branch.instruct()` ‚Üí `branch.operate()`

```text
AttributeError: 'Branch' object has no attribute 'instruct'
```

```bash
grep -rn "\.instruct(" --include="*.py" .
```

```python
# before: await branch.instruct(instruction="...", stream_persist=True, persist_dir="./runs")
# after:  await branch.operate( instruction="...", stream_persist=True, persist_dir="./runs")
```

- ‚úì Rename `.instruct(` ‚Üí `.operate(`; all kwargs unchanged; `Instruct(...)` objects unchanged
- ‚úì New optional `middle=` kwarg ‚Äî omit to auto-dispatch (API‚Üí`communicate`, CLI‚Üí`run_and_collect`)

## ¬ß2 `lionagi.operations.instruct` module removed

```text
ModuleNotFoundError: No module named 'lionagi.operations.instruct'
```

```bash
grep -rn "from lionagi.operations.instruct\|import instruct" --include="*.py" .
```

```python
from lionagi.operations.instruct import instruct  # 0.22.5
from lionagi.operations.operate  import operate   # 0.22.6
```

- ‚úì `Instruct` dataclass unchanged ‚Äî only module path and function name changed

## ¬ß3 Persistence layout: `logs/agents/` ‚Üí `runs/{run_id}/`

```text
# 0.22.5: ~/.lionagi/logs/agents/{provider}/{branch_id}.json

# 0.22.6: ~/.lionagi/runs/{YYYYMMDDTHHMMSS-uuid6}/
    ‚îú‚îÄ‚îÄ run.json
    ‚îú‚îÄ‚îÄ branches/{branch_id}.json
    ‚îú‚îÄ‚îÄ stream/{branch_id}.buffer.jsonl
    ‚îî‚îÄ‚îÄ artifacts/{agent_id}/
```

`li agent -r {branch_id}` unchanged ‚Äî `find_branch()` falls back to `logs/agents/` for pre-0.22.6 branches.

```bash
grep -rn "logs/agents" --include="*.py" .
```

- ‚úì Replace `~/.lionagi/logs/agents/{provider}/` references ‚Üí `~/.lionagi/runs/*/branches/`
- ‚úì Pass `LIONAGI_RUN_ID=$run_id` when spawning CLI subprocesses to inherit the parent run

## Minor changes (¬ß4‚Äì¬ß8)

**¬ß4 `last_branch.json` schema** ‚Äî auto-normalized; direct parsers only.
`{"provider": "...", "branch_id": "..."}` ‚Üí `{"run_id": "...", "branch_id": "..."}`.
Grep: `grep -rn "last_branch.json" --include="*.py" .`

- ‚úì Replace `data["provider"]` with `data["run_id"]` in direct JSON parsers

**¬ß5 Flat artifact dumps** ‚Äî path format changed.
`./artifacts/r1_research.md` ‚Üí `./artifacts/r1/research.md`
Grep: `grep -rn "artifact" --include="*.py" . | grep "_\.md"`

- ‚úì Update: `{id}_{name}.md` ‚Üí `{agent_id}/{name}.md`

**¬ß6 DAG op name** ‚Äî `"instruct"` ‚Üí `"operate"` in `run.json` manifests; end-user CLI unaffected.
Grep: `grep -rn '"instruct"' --include="*.py" --include="*.json" .`

- ‚úì Replace `operation.type == "instruct"` with `"operate"` in custom flow plugins
**¬ß7 `"codex/gpt-5.4"` fallback** ‚Äî `ValueError: Provider must be provided` if model unset.

- ‚úì Add `--model codex/gpt-5.4` or `export LIONAGI_CHAT_MODEL=codex/gpt-5.4`
**¬ß8 `TEAM_WORKER_SYSTEM`** ‚Äî now `BARE_WORKER_SYSTEM + TEAM_COORD_SECTION`; may double-include base.
Grep: `grep -rn "TEAM_WORKER_SYSTEM\|TEAM_COORD_SECTION" --include="*.py" .`

- ‚úì Use `TEAM_COORD_SECTION` for coordination-only block; audit custom prompts for duplicate base
*See also: [`CHANGELOG.md`](https://github.com/ohdearquant/lionagi/blob/main/CHANGELOG.md) ‚Äî 0.22.6 entry.*
